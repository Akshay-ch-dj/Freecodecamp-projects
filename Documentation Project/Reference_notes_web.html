Javascript

Javascript(JS in short) is a cross-platform, object-oriented scripting language. In web development it is extensively used to add interactive functionality to the web application. All modern browsers run javascript directly( can see it from the "inspect(ctrl+shift+I)" element from the chrome browser), or can run as a full ".js" script connected to an html file. Logic applied through JS is used to change the HTML/CSS properties shown on the page, thereby adds more control and interaction to web pages.

According to StackOverflow 2020, JavaScript is the fastest growing and most popular programming language. It can used in web, in mobile applications with react native, desktop applications with electron(applications like VS code, atom, whatsapp for desktop uses electron framework).

There are different versions of JS, the recent and widely supported version is ES6.

To start simple scripting in JS, directly go to the chrome's(or any modern browsers) inbuilt console [on chrome: inspect_element console, or just type F12]. Don't click on this button to get alerts.

Code editor perks

Visual Studio Code is an amazing code editor from microsoft, ie open source, comes with great community support and great extensions.

VS code extensions

These are the extensions I'm personally using and its settings.

    Visual Extensions

  1. Prettier: formats the code make it prettier

    In settings(ctrl+,), search for "format on save": check it to automatically format on every save. Also, there is lot of settings that can be set based on personal preferences.

  2. Bracket Pair Colorizer 2: The version 2 of em.

    The version 2 uses same parsing engine of VS code, that increases speed and accuracy.

  3. Indent-rainbow: Add colors to the margin.

    Functional Extensions

  1. Auto Rename Tag: Change the beginning tag, end gets automatically changed (html/xml)

  2. Code Spell Checker: Spelling checker for source code

    It comes handy when editing html, for quick suggestions use: ctrl+.

  3. HTML CSS support: class completions to html

    For eg.: To add a new div with the already defined class name("my_classname"), just start typing ".my-" then the class will be in the suggestions, hit "tab" to create automatically.

  4. Rest Client: See the backend API responses

    Similar to the "postman" program. make a http file and type in the requests, then send the request, that automatically opens the response window on side with all the details and returned data. Also can have multiple requests(separate them with triple hash tags(###)).

  5. Live Server: Creates a Local dev server with live reloading

    From any html/template files hit the "Go live" button on bottom, it automatically refreshes the new content.

  6. CSS peek: peek the css definitions from html file.

    Just hover over a class/id to peak the css, click it to go directly to that css class(in a new temporary window), multiple definitions also shows up.

  7. Live Sass Compiler: will compile sass to css

    Go to settings of live sass compiler, locate "liveSassCompile.settings.formats:" in the json, for production environment set the format to compressed, extensionName to .min.css going to minify and compress the css files automatically. Can also set the savePath (/dist/css), also set the liveSassCompile.settings.generateMap to false.

    When a new "scss" file created with sass styling, then hit the "watch sass" button on bottom, it compiles the sass into normal css and save it in the set path(here: on /dist/css/)

VS code emmets

Emmet helps drastically speed up the rate of writing the markup.

  • "!": just type the exclamation mark and hit tab to create an html5 boilerplate automatically. Change the contents in the highlighted areas using tab to switch.
  • To create an "ul" skelton with 5 "li" elements: use ul>li*5
  • button#email-me: To create "button" with id="email-me", similarly p.main-content: creates a paragraph with class="main-content"
  • lorem30: Just gives the first 30 words of lorem ipsum.

Useful Web tools & Services

Handy Web Tools

These are some web tools that comes handy at times,

Bundlephobia

It just helps to analyze how much an npm package will add to your bundle. It works with all modern frontend frameworks. Just enter the package name and analyze how much the addition of it will cost to your bundle size, download time, tree-shakable or not and a bunch of other information, it is available for every version of that package, can even upload entire package to see the size comparison and other info.

Cloudcraft

It is officially for AWS, but it can get useful for other services too, great for planning and budgeting the project. It's got a drag and drop interface to connect different cloud infrastructure services together, gives the estimated breakdown cost of every services.(the pricing almost translate to azure, GCP well).

Figma

A web alternate to Adobe Illustrator, Figma is a design tool that helps draw graphics and create markups for the app, available as web app and desktop app. The web app also gives great performance(uses Web Assembly technology). For a web developer it can extract the styles as CSS and apply different properties realtime. The free version itself offers great productivity.

FontFlipper

Helps selecting the free fonts from google in an intuitive way. One can also bring in their own graphics and experiment to get the best results. The focussed and simple UI with select what you like and download later directly from google fonts.

Rive

Another design tool, but it is for rendering 2D vector animations, with keyframes to control the motion of the graphics. It is a part of open design movement, see others works,learn from it, it can also export the animations directly to flutter

VisBug: Chrome Extension

A better implemented inspect element system. Using this chrome extension, can change the way you design the page. Its like backward designing, change the properties the size, position, scale and copy the final css back after all modifications. It needs some learning to work well on it but every second worth.

Insomnia

Backend tool, helps to debug API's
